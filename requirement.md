Buatkan program sederhana Link Shortener 
dengan PHP native tanpa framework apapun.
Boleh install composer sebagai dependency.
Gunakan database SQL lite saja agar tidak repot membuat database lagi.

Kebutuhan :
1. Halaman administratif pembuatan shortlink simple saja tampilannya clean, white, elegant, dengan tailwind css CDN.
2. Daftar halaman : Login, dashboard home, statistik link dan ref.
3. Gunakan pola domain.com/s/<slug> kemudian arahkan ke link utama
4. Bisa tulis custom cantik sendiri, jika blm dipake orang, biar engga random banget.
5. Ketika klik link langsung direct saja, tanpa basa basi iklan, immediatly.

Buat proyek ini simple saja gak udh pake kode clean segala, semuanya native php misal index.php short.php
